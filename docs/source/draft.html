Residue Similarity Score

The residue similarity score involves Mean Distance (MD) scores, inspired by Thompson et al. (2001). This score is calculated for each column as follows:

  1. Define a substitution matrix containing the scores of changing each residue for the others: By default, trimAl employs the BLOSUM62 matrix for amino acid sequences and the identity matrix for nucleotide residues. However, users can input alternative matrices. The Euclidean distance measures dissimilarity between residues based on their substitution scores.

In trimAl v1.2, if the gap score for the column is equal to or less than 0.2, the MD\text MDMD﻿ score is set to zero. This adjustment penalizes columns with numerous gaps, preventing those with few residues from receiving inflated scores. This penalty was introduced to address potential issues when trimming alignments based solely on similarity information.
